隐式转换

  • Number 转 String

    使用运算符 + 连接一个空字符串.

  • String 转 Number

    使用运算符 * 乘以 1.

  • Number/String 转 Boolean

    使用!运算符(取反).

    • String 转 Boolean

在 if 代码块中，去掉大括号

  • 单行去掉大括号

    if条件表达式后面若只有一句代码，则大括号自动省略。

  • 多行去掉大括号

    if条件表达式后面若有多行代码，要省略大括号，则每句代码后用 , 隔开。
